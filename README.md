# HTMLSkeleton

## Code

```
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Page Title</title>
    <meta name="viewport" content="width=device-width,initial-scale=1">
    <link rel="stylesheet" href="./style.css">
    <script src="./script.js"></script>
  </head>
  <body>

    <!-- main body -->

  </body>
</html> 
```
